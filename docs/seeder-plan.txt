## Section 2 — Shared Helper

### `Shared/HistoryHelper.php`

Namespace `Database\Seeders\Shared`. A plain class, not a seeder. Contains static methods used by both history seeders to avoid duplication.

**Static methods:**

`makeTokenPhysicalId(int $index): string`
Converts a 0-based index to a physical ID. 0–8 = A1–A9, 9–17 = B1–B9, etc.
```php
$letter = chr(65 + intdiv($index, 9));
$number = ($index % 9) + 1;
return $letter . $number;
```

`makeQrHash(string $siteSlug, string $physicalId): string`
Returns `hash('sha256', 'TOKEN_' . $siteSlug . '_' . $physicalId)`.

`spreadSessionsAcrossDay(int $count, Carbon $baseDate): array`
Returns an array of `$count` Carbon timestamps spread across a workday (8:00 AM–4:30 PM) with a morning peak. Distribution: 8–11 AM = 50%, 1–3 PM = 35%, 3–4:30 PM = 15%. Returns sorted ascending.

Implementation:
```php
$morning = (int) round($count * 0.50);
$afternoon = (int) round($count * 0.35);
$lateAfternoon = $count - $morning - $afternoon;

$times = [];
for ($i = 0; $i < $morning; $i++) {
    $times[] = $baseDate->copy()->setTime(8, 0)->addMinutes(rand(0, 179));
}
for ($i = 0; $i < $afternoon; $i++) {
    $times[] = $baseDate->copy()->setTime(13, 0)->addMinutes(rand(0, 119));
}
for ($i = 0; $i < $lateAfternoon; $i++) {
    $times[] = $baseDate->copy()->setTime(15, 0)->addMinutes(rand(0, 89));
}
sort($times); // sort Carbon objects — use usort with $a->lt($b)
return $times;
```

`pickCategory(string $trackName): string`
If trackName contains 'Priority': return 'Senior Citizen' 60% / 'PWD' 40%.
If trackName is 'Regular': return 'Regular'.

`pickOutcome(int $dayIndex): string`
Returns 'completed', 'no_show', or 'cancelled' based on weighted random.
```php
$roll = rand(1, 100);
if ($roll <= 75) return 'completed';
if ($roll <= 85) return 'no_show';
return 'cancelled';
```

`insertLog(int $sessionId, ?int $stationId, ?int $staffId, string $actionType, ?int $prevStationId, ?int $nextStationId, string $createdAt): void`
Inserts one row into `transaction_logs` using `DB::table()->insert()`.
```php
DB::table('transaction_logs')->insert([
    'session_id'          => $sessionId,
    'station_id'          => $stationId,
    'staff_user_id'       => $staffId,
    'action_type'         => $actionType,
    'previous_station_id' => $prevStationId,
    'next_station_id'     => $nextStationId,
    'remarks'             => null,
    'metadata'            => null,
    'created_at'          => $createdAt,
]);
```

$step1Out  = $step1In + rand(4, 12) minutes

$step2Out  = $step2In + rand(4, 12) minutes
$step3In   = $step2Out + 1 minute
$step3Out  = $step3In + rand(4, 12) minutes

Do NOT insert one by one. Collect all sessions for a day into an array and do one bulk insert:
```php
DB::table('queue_sessions')->insert($sessionRows);
```

After bulk insert, retrieve the inserted IDs. On SQLite this requires fetching by `token_id + started_at` since `lastInsertId()` only returns the last one. Simplest approach: after inserting a day's sessions, query them back:
```php
$insertedSessions = DB::table('queue_sessions')
    ->where('program_id', $aics->id)
    ->whereDate('started_at', $baseDate->toDateString())
    ->orderBy('id')
    ->get(['id', 'token_id', 'started_at', 'status', 'track_id', 'current_station_id']);
```

**Transaction logs per session:**

Collect all log rows for all sessions in the day into one array, insert bulk at the end of the day.

For each `$session` from `$insertedSessions`, match it back to your in-memory session plan (by `token_id + started_at`) to get the computed timestamps.

Log rows by outcome:

**Completed (4 steps):**
```
bind         station=null,       staff=$admin->id,    action=bind,       at=$bindAt
call         station=$stations[0], staff=$staff[0],    action=call,       at=$callAt
check_in     station=$stations[0], staff=$staff[0],    action=check_in,   at=$step1In
transfer     station=$stations[0], staff=$staff[0],    action=transfer,   at=$step1Out,  next=$stations[1]
check_in     station=$stations[1], staff=$staff[1],    action=check_in,   at=$step2In
transfer     station=$stations[1], staff=$staff[1],    action=transfer,   at=$step2Out,  next=$stations[2]
check_in     station=$stations[2], staff=$staff[2],    action=check_in,   at=$step3In
transfer     station=$stations[2], staff=$staff[2],    action=transfer,   at=$step3Out,  next=$stations[3]
check_in     station=$stations[3], staff=$staff[3],    action=check_in,   at=$step4In
complete     station=$stations[3], staff=$staff[3],    action=complete,   at=$completedAt
```

**No-show:**
```
bind         station=null,       staff=$admin->id,    action=bind,       at=$bindAt
call         station=$stations[0], staff=$staff[0],    action=call,       at=$callAt
no_show      station=$stations[0], staff=$staff[0],    action=no_show,    at=$callAt+12min
```
Set session `completed_at` to `$callAt+12min`. Set `current_station_id` to `$stations[0]->id`. Set `status=no_show`.

**Cancelled (cancelled at station 1 or 2):**
```
bind         action=bind,   at=$bindAt
call         action=call,   at=$callAt,   station=$stations[0]
check_in     action=check_in, at=$step1In, station=$stations[0]

```
Set session `completed_at` to the cancel time. Set `status=cancelled`. Set `current_station_id` to `$stations[0]->id`.

Bulk insert all log rows at end of each day:
```php
DB::table('transaction_logs')->insert($logRows);
```

**After all sessions across all days are inserted, reset token statuses:**
```php
DB::table('tokens')->where('site_id', $site->id)->update([
    'status'             => 'available',
    'current_session_id' => null,
    'updated_at'         => now(),
]);
```

**Important: `started_at` vs `queued_at_station` vs wait time.**

The analytics `getWaitMinutesBySessionId` computes wait as `started_at` → `MIN(call log created_at)`. So `started_at` must be the bind time and the `call` log timestamp must be bind time + wait minutes. This is exactly what the plan above does. The wait time distribution will show values between 2–8 minutes, which hits all buckets from `0–5 min` and `5–10 min`.

        'override_steps'         => null,
        'identity_registration_id' => null,
        'created_at'             => $yesterday->copy()->addMinutes($i * 5)->toDateTimeString(),
        'updated_at'             => $yesterday->copy()->addMinutes($i * 5)->toDateTimeString(),
    ]);
    $sessionId = DB::getPdo()->lastInsertId();
    // Insert bind log only — they were queued but never called
    HistoryHelper::insertLog($sessionId, null, $admin->id, 'bind', null, null,
        $yesterday->copy()->addMinutes($i * 5)->toDateTimeString());
}
```

Also update these 3 tokens to `status=in_use` and `current_session_id` pointing to their session:

// After inserting each stale session, update the token
DB::table('tokens')->where('id', $token->id)->update([
    'status'             => 'in_use',
    'current_session_id' => $sessionId,
    'updated_at'         => now(),

```
